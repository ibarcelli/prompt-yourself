# Prompt Yourself — Decisions

## Decisiones Canónicas
No se reabren sin aprobación explícita del Human Lead (Ives).

### DEC-001: Lenguaje de los prompts
- **Decisión**: Primera persona, presente simple declarativo. "Yo recibo", no "estoy recibiendo".
- **Razón**: Funciona como instrucción al sistema operativo interno, no como narrativa.
- **Fecha**: 2026-03-23

### DEC-002: Formato dual científico/metafísico
- **Decisión**: La app tiene dos marcos de lenguaje. Metafísico (campo cuántico, activación, manifestación) y científico (neuroplasticidad, priming cognitivo, ondas cerebrales). El mecanismo subyacente es el mismo.
- **Razón**: Persona Fiorella (escéptica) necesita frame científico. Persona Gisella (practicante) necesita frame metafísico. No se diluye ninguno.
- **Fecha**: 2026-03-23

### DEC-003: MVP es texto, no audio
- **Decisión**: El MVP entrega texto para leer en voz alta. Audio con frecuencias binaurales es fase posterior.
- **Razón**: Audio bien hecho requiere producción. Texto leído por el propio usuario tiene efecto de priming más fuerte por involucrar voz y aparato motor.
- **Fecha**: 2026-03-23

### DEC-004: Precio preliminar
- **Decisión**: ~$20 MXN/mes. Gratis para primeros usuarios. Precio final se define con POC funcionando.
- **Razón**: Accesible para todos. La sofisticación se financia con ingresos.
- **Fecha**: 2026-03-23

### DEC-005: Stack técnico
- **Decisión**: PWA (web app progresiva) con builder no-code/low-code + API de IA (Claude/GPT) para generación de prompts.
- **Razón**: Sin presupuesto para desarrollo complejo, sin capacidad de coding. Arrancar ligero y escalar.
- **Fecha**: 2026-03-23

### DEC-006: Ives es usuario cero
- **Decisión**: Ives usa la app como primer proof of concept. Su goal alignment document es modelo de input.
- **Razón**: Autenticidad. Si funciona para el fundador, funciona.
- **Fecha**: 2026-03-23

### DEC-007: Simplicidad radical
- **Decisión**: Sin gamificación, sin menús complejos, sin features innecesarios. Abres, recibes, practicas, cierras.
- **Razón**: Declaración explícita de Ives: "bastante bastante simple".
- **Fecha**: 2026-03-23
